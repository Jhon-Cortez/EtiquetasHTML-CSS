Formularios

En HTML los formularios sirven para recoger informacion del usuario y enviarla o procesarla. Esta sección va dirigida a los formularios, cada uno con su respectiva explicación

  • form: agrupar campos y enviar datos.
  • label: poner el nombre del campo y enlazarlo al input.
  • input: capturar datos del usuario.
  • textarea: escribir texto largo.
  • select: escoger una opción de una lista.
  • option: definir cada opción dentro del select.
  • button: crear un botón.
  • fieldset: agrupar campos relacionados.
  • legend: poner el título del fieldset.

    • Input recomendados

  • text: sirve para escribirtexto normal
  • email: sirve para ingresar un correo y valida un formato básico
  • password: sirve para contraseñas, el texto se oculta con puntos o asteriscos
  • number: sirve para números; permite límites con min y max
  • date: sirve para seleccionar una fecha con calendario
  • radio: sirve para escoger una sola opción entre varias
  • checkbox: sirve para marcar opciones
  • file: sirve para subir archivos
  • submit: sirve para enviar el formulario
