Textos y Contenidos

En esta sección vamos a ver algunas etiquetas básicas de HTML para textos y contenidos.

  • h1 a h6:

    sirven para títulos y subtítulos (h1 es el más importante, h6 el menos), se puede evidenciar en la imagen que es el ejemplo de como aplicar el h1, es la misma estructura para los demas h.

  • p:

    sirve para escribir párrafos de texto, se aplica asi:
  • span: sirve para envolver texto en línea y poder aplicarle estilos o manipularlo.
  • strong: sirve para marcar texto como importante.
  • em: sirve epara dar énfasis a una palabra o frase.

  • br:
    sirve para hacer un salto de línea.
  • hr: sirve para poner un separador entre partes del contenido.
